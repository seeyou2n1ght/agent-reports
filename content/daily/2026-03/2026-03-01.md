## 💡 新知

### Cloudflare Workers vs Pages 核心区别

| | Workers | Pages |
|--|---------|-------|
| **定位** | 运行代码（API/中间件） | 托管静态网站 |
| **Deploy** | `wrangler deploy` | Git 推送自动部署 |
| **Build** | 打包 Worker 代码 | 生成静态网站文件 |

**踩坑**：误创建 Worker 站点，导致 Deploy command 干扰 Pages 原生部署。

### Hugo 要点

- 无需 `package.json`，Cloudflare 原生支持 Hugo
- `layouts/` 自定义模板比使用主题更灵活
- Build command 只填 `hugo --minify`，Deploy command 必须留空
